개발 환경 ==> win 11
---

기술 스택 ==>
Frontend	React, Vite, JavaScript
Backend	Java, Spring Boot
Database	-
Tools		Git, STS, VS Code

---

개발목표 ==>
맥주 양조 레시피 시뮬레이터 돌리는 풀스택 웹 애플리케이션
맥주 양조 시 필요한 재료들 데이터 베이스를 가지고, 사용자가 입력값만 넣으면 어떤 맥주가 완성될 지 계산하여 알려주는 프로그램이 목표

---
주요 기능 ==>
양조 시뮬레이터 : 현재 맥주 원재료 투입 파라미터에 대하여 시간별, 공정별 워트 상태(abv, ibu, og 등) 계산하여서 알려줌
UI : Vite를 활용해서 빠른 렌더링

---

cd backend
./mvnw spring-boot:run

cd frontend
npm install
npm run dev
